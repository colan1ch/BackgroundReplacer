# Video Background Replacer & Face Identifier

Проект для Mac / iOS (SwiftUI), который в реальном времени заменяет фон на видеопотоке и параллельно **идентифицирует (распознает)** лица людей, сверяя их с локальной базой данных («галереей известных лиц»).

## 🚀 Что умеет приложение?

1. **Интерактивная замена фона (Сегментация)** Использует Apple Vision (`VNGeneratePersonSegmentationRequest`) для отделения силуэта человека. Доступно 3 разных фоновых цикличных видео (Лес, Город, Офис). Если фон отключен — показывается стандартная картинка с камеры.
2. **Распознавание лиц в реальном времени (Идентификация)** Вместо обычного трекинга силуэтов приложение ищет лица через `VNDetectFaceRectanglesRequest`, вырезает их и отправляет в нейросетевую модель `CoreML`.
3. **Работа со списками и базой фоток** * Если лицо совпало с базой (например, Мао Цзэдун) — рамка подсвечивается **зеленым**, сверху пишется имя и процент уверенности нейросети.
   * Если человека нет в базе — рамка подсвечивается **красным**, а в статусе пишется *«Не определен»* (или *Неизвестный пользователь*).

## 🛠 Архитектура проекта

Приложение написано на **SwiftUI** с использованием фреймворка **Observation** (iOS 17+ / macOS 14+):

* `CameraManager.swift` — отвечает за захват видеопотока с камеры (AVFoundation), настройку сессии, переключение между фронтальной/основной камерой и пропуск кадров для оптимизации нагрузки.
* `VideoPlayerController.swift` — кастомный плеер (`AVQueuePlayer` + `AVPlayerLooper`), который крутит зацикленные фоновые видео без швов и рывков.
* `BackgroundReplacementProcessor.swift` — «мозг» приложения. Здесь параллельно работают:
  * Запрос на сегментацию людей (`CIBlendWithMask`) для хромакея.
  * Вырезание лиц по координатам CoreImage с учетом перевернутой оси Y во фреймворке Vision.
  * Классификация вырезанного лица через CoreML-модель.
* `ContentView.swift` — интерфейс: выводит видеопоток, рисует bounding-box'ы, плашки с именами и панель переключения фонов.

## 📂 Как настроить свою базу фоток (Create ML)

Чтобы приложение узнавало вас или преподавателя:

1. Создайте папку `FaceBase`, внутри неё подпапки с именами (например, `Mao_Zedong`, `Prepod_Ivanov`) и одну папку `Unknown` для случайных людей.
2. Закиньте в каждую папку по 15-20 фотографий (желательно крупно лица с разных ракурсов).
3. Откройте **Xcode -> Open Developer Tool -> Create ML**. Выберите **Image Classification**.
4. Перетащите папку `FaceBase` в поле *Training Data* и нажмите кнопку **Train** (▶️).
5. Скомпилированную модель `MyFaceClassifier.mlmodel` перетащите прямо в проект Xcode.

## 💻 Системные требования

* **Язык**: Swift (SwiftUI)
* **Платформа**: iOS 17.0+ / macOS 14.0+
* **Фреймворки**: Vision, CoreML, CoreImage, AVFoundation
* *Важно*: На симуляторах Xcode сегментация и CoreML могут не работать из-за отсутствия физической камеры и Neural Engine, проект тестировался и работает на реальном устройстве.
